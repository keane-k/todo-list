# To-Do List
A To-Do List designed around Destiny 2 theme.

User's data is saved locally, refreshing the page or re-opening the page will not result in any data loss, they'll just continue right where they left off.

## How to Use
1. Users can type out their task on the searchbar `Add your text`
2. Click on the button `Add`

## To Check a task
1. Click on the Unchecked-Square (☐) to become Checked-Square (☑)

## To Uncheck a task
1. Click on the Checked-Square (☑) to become Unchecked-Square (☐)
   
## To Remove a task from the List
1. Click on the Cross (✖) to remove task from List
---
## How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
